# 运输服务金融产品收益计算模型

一个纯前端的金融产品收益测算工具，支持多方案对比、单车/渠道/毛利三维度分析，以及可视化图表展示。

## 功能概览

- **总览看板** — 核心 KPI 与收益公式一览
- **产品方案** — 多方案管理，含面客收费、运营成本、渠道费用三组配置
- **收益测算** — 单车模型（回本周期）、渠道模型（逻辑斯蒂增长曲线）、毛利模型（3年总毛利）
- **情景对比** — 多方案参数横向对比
- **数据管理** — 基准数据表维护

## 技术栈

- 纯 HTML/CSS/JS 单文件，无需构建
- [Chart.js](https://www.chartjs.org/) — 图表渲染
- [SheetJS](https://sheetjs.com/) — Excel 导出
- CSS 自定义属性设计系统（色彩/排版/间距/圆角/阴影）
- localStorage 持久化 + JSON 导入导出

## 快速开始

直接用浏览器打开 `index.html` 即可使用，也可以通过 GitHub Pages 在线访问。

```bash
# 本地预览（任选一种）
python -m http.server 8080
npx serve .
```
