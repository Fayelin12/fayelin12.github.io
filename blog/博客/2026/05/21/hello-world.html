这是我的第一篇博客文章。

今天成功搭建了这个基于 Jekyll + GitHub Pages 的博客系统，并通过 GitHub Actions 实现了自动部署。博客源码保存在私有仓库中，构建后的静态页面会自动推送到 GitHub Pages 主仓库的 /blog 子目录下。

整个过程包括：

  • 安装 Ruby、Jekyll 和 Bundler
  • 初始化 Jekyll 项目并配置 baseurl
  • 创建 GitHub Actions 工作流实现自动构建和部署
  • 配置 Personal Access Token 实现跨仓库推送

期待在这里记录更多技术学习和生活感悟！
